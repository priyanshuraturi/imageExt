## Face Detection Open-CV

A Python Project That Exracts Images With a Human Face From a Directory And Copies Them To Another Directory.

## How To use:

```bash
python imgext.py path_of_Folder_with_images path_of_result_folder
```

Just Run The Following Script and Pass The Respective Path in Arguments

## Requirements:
OpenCV2
